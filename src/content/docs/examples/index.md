## Running Examples

All examples require Go 1.21+ and goctl installed.

```bash
go install github.com/zeromicro/go-zero/tools/goctl@latest
```

Clone the official example repository:

```bash
git clone https://github.com/zeromicro/go-zero.git
cd go-zero/example
```
