面向任务的实践指南，涵盖 go-zero 开发中的各类操作。每个指南从头到尾带你完成一项具体任务。

## 安装

搭建开发环境：安装 Go、goctl、protoc 和 IDE 插件。

## 创建项目

通过命令行、GoLand 或 VS Code 创建新的 go-zero 项目。

## HTTP 服务

使用 goctl 和 go-zero 构建并运行 HTTP 服务。

## gRPC 服务

定义 protobuf schema、生成代码，运行 gRPC 服务端和客户端。

## CLI 任务

使用 `goctl` 工具：生成 API/RPC 代码、格式化 `.api` 文件、生成 MongoDB 和 MySQL 模型。

## DSL

使用 go-zero 的 `.api` 和 `.proto` DSL 语法。

## 数据存储

连接并查询 MySQL、MongoDB 和 Redis。

## 消息队列

使用消息队列和延迟队列。

## 日志

配置和使用 go-zero 的结构化日志。

## 静态配置

加载和管理静态配置文件。

## 内存缓存

使用进程内缓存。

## 配置中心

与远程配置中心集成。
