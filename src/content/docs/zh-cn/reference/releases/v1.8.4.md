发布于 **2025-06-08** — [GitHub Release](https://github.com/zeromicro/go-zero/releases/tag/v1.8.4)

## Highlights

- **Continuous profiling** support ([#4867](https://github.com/zeromicro/go-zero/pull/4867)).
   ```yaml
   Profiling:
      ServerAddr: http://<pyroscope-server>:<port>
  ```
- **Embedded File Server Enhancements**: Added support to serve files using `embed.FS`, simplifying static file serving ([#4847](https://github.com/zeromicro/go-zero/pull/4847), [#4851](https://github.com/zeromicro/go-zero/pull/4851)).
- **Performance Optimizations**: Improved performance for hashing and slicing operations ([#4891](https://github.com/zeromicro/go-zero/pull/4891), [#4877](https://github.com/zeromicro/go-zero/pull/4877)).

## What's Changed

- **New Features**:
    - File server now supports `embed.FS` ([#4847](https://github.com/zeromicro/go-zero/pull/4847)).
    - Continuous profiling support ([#4867](https://github.com/zeromicro/go-zero/pull/4867)).
- **Optimizations**:
    - Improved slicing and comparison using Go standard library functions ([#4841](https://github.com/zeromicro/go-zero/pull/4841), [#4840](https://github.com/zeromicro/go-zero/pull/4840), [#4877](https://github.com/zeromicro/go-zero/pull/4877), [#4879](https://github.com/zeromicro/go-zero/pull/4879)).
    - Time formatting standardized with the Go standard library ([#4875](https://github.com/zeromicro/go-zero/pull/4875)).
    - Hashing performance improvements ([#4891](https://github.com/zeromicro/go-zero/pull/4891)).
