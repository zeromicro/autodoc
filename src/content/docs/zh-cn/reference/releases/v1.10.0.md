发布于 **2026-02-15** — [GitHub Release](https://github.com/zeromicro/go-zero/releases/tag/v1.10.0)

## 🎉 Highlights

This release brings Go 1.23 support, MCP SDK migration, and several important bug fixes including race condition resolutions.

## ✨ New Features

- **Go 1.23 Support**: Upgraded minimum Go version to 1.23 ([#5359](https://github.com/zeromicro/go-zero/pull/5359))
- **MCP Framework**: Migrated to official go-sdk with simplified API ([#5362](https://github.com/zeromicro/go-zero/pull/5362))
- **Gateway Enhancement**: Exported `WithDialer` option for custom gRPC client configuration ([#5406](https://github.com/zeromicro/go-zero/pull/5406))

## 🐛 Bug Fixes

- **Circuit Breaker**: Fixed context cancellation incorrectly triggering breaker in httpc ([#5360](https://github.com/zeromicro/go-zero/pull/5360))
- **Service Discovery**: Resolved data race in service discovery map access ([#5408](https://github.com/zeromicro/go-zero/pull/5408))
- **Configuration**: Fixed support for equal signs in property values ([#5392](https://github.com/zeromicro/go-zero/pull/5392))
- **Configuration**: Removed redundant validation ([#5372](https://github.com/zeromicro/go-zero/pull/5372))

## 🔧 Improvements

- **MCP Routes**: Refactored routes and hardened AddTool implementation ([#5375](https://github.com/zeromicro/go-zero/pull/5375))
- **Testing**: Added comprehensive validation tests for Load function ([#5388](https://github.com/zeromicro/go-zero/pull/5388))

## 🗑️ Deprecations

- **Jaeger Exporter**: Removed due to official deprecation ([#5361](https://github.com/zeromicro/go-zero/pull/5361))

## 📦 Dependencies

- Bumped `go.mongodb.org/mongo-driver/v2` from 2.4.1 to 2.5.0 ([#5385](https://github.com/zeromicro/go-zero/pull/5385), [#5393](https://github.com/zeromicro/go-zero/pull/5393))
- Bumped `github.com/alicebob/miniredis/v2` from 2.35.0 to 2.36.1 ([#5381](https://github.com/zeromicro/go-zero/pull/5381), [#5386](https://github.com/zeromicro/go-zero/pull/5386))
- Bumped `github.com/redis/go-redis/v9` from 9.17.2 to 9.17.3 ([#5390](https://github.com/zeromicro/go-zero/pull/5390))
- Bumped `github.com/modelcontextprotocol/go-sdk` from 1.2.0 to 1.3.0 ([#5413](https://github.com/zeromicro/go-zero/pull/5413))
