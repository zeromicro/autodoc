发布于 **2025-07-12** — [GitHub Release](https://github.com/zeromicro/go-zero/releases/tag/v1.8.5)

## Features

- **SQL Read/Write Splitting**: Introduced SQL read/write splitting for improved database performance and scalability ([#4976](https://github.com/zeromicro/go-zero/pull/4976), [#4990](https://github.com/zeromicro/go-zero/pull/4990), [#5000](https://github.com/zeromicro/go-zero/pull/5000)).
- **Serverless Support in REST**: Added support for serverless use in REST services ([#5001](https://github.com/zeromicro/go-zero/pull/5001)).

## Bug Fixes & Improvements

- Fixed HTTP SSE method timeout not working when timeout is set by server ([#4932](https://github.com/zeromicro/go-zero/pull/4932))
- Fixed timeout `0s` not working in API files ([#4932](https://github.com/zeromicro/go-zero/pull/4932))
- Fixed panic caused by `time.Duration` type with numerical values ([#4944](https://github.com/zeromicro/go-zero/pull/4944))
- Fixed duration type comparison in environment variable processing ([#4979](https://github.com/zeromicro/go-zero/pull/4979))
