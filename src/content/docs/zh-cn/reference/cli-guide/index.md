## 概览

`goctl` 是 go-zero 的代码生成工具，可以脚手架 HTTP 服务、gRPC 服务、Model、Dockerfile、Kubernetes 清单等。

- [RPC 生成](rpc/) — 从 `.proto` 文件生成 gRPC 服务
- [Model 生成](model/) — 数据库 Model 代码
- [Docker](docker/) — 生成 Dockerfile
- [Kubernetes](kube/) — 生成 K8s 清单
- [环境](env/) — 检查构建环境

- [迁移](migrate/) — 迁移模板
- [Shell 补全](completion/) — 自动补全配置
