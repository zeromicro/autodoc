欢迎加入 go-zero 社区！本节包含贡献指南、常见问题与社区资源。

## 如何贡献

go-zero 是一个开源项目，欢迎各种形式的贡献。

- **报告 Bug** — 创建 [GitHub issue](https://github.com/zeromicro/go-zero/issues)
- **修复 Bug** — 找到 `good first issue` 标签并提交 PR
- **添加特性** — 先在 issue 中讨论方案，再实现
- **改进文档** — 通过任意页面上的"编辑页面"链接编辑本站
- **分享示例** — 添加到[示例](../examples/)章节

### 快速上手

1. [Fork](https://github.com/zeromicro/go-zero/fork) 仓库
2. 克隆你的 fork：`git clone https://github.com/<you>/go-zero.git`
3. 创建分支：`git checkout -b fix/typo-in-readme`
4. 修改代码，运行测试：`go test ./...`
5. 推送并创建 Pull Request
