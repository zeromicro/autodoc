## 概述

goctl upgrade 用于升级 goctl 工具。

## goctl upgrade 指令

goctl upgrade 会将当前 goctl 工具升级到最新版本。

```bash
$ goctl upgrade --help
Upgrade goctl to latest version

Usage:
  goctl upgrade [flags]

Flags:
  -h, --help   help for upgrade
```
