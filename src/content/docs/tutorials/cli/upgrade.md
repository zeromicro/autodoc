## Overview

goctl upgrade for upgrading goctl tools.

## goctl upgrade directive

goctl upgrade will upgrade the current goctl tool to the latest version.

```bash
$ goctl upgrade --help
Upgrade goctl to latest version

Usage:
  goctl upgrade [flags]

Flags:
  -h, --help   help for upgrade
```
