Released on **2025-04-06** — [GitHub Release](https://github.com/zeromicro/go-zero/releases/tag/v1.8.2)

We're pleased to announce the release of go-zero v1.8.2. This release includes new features, bug fixes, dependency updates, and code improvements.

## 🚀 New Features

- **SSE Support**: Added `rest.WithSSE` to build Server-Sent Events routes more easily ([#4729](https://github.com/zeromicro/go-zero/pull/4729))
- **Redis Command**: Added support for Redis `GETDEL` command ([#4709](https://github.com/zeromicro/go-zero/pull/4709))
- **HTTP Client Enhancement**: Added support for serialization of anonymous fields in HTTP client (httpc) ([#4676](https://github.com/zeromicro/go-zero/pull/4676))

## 🐛 Bug Fixes

- **PostgreSQL Data Types**: Fixed issues with numeric/decimal data types in PostgreSQL ([#4686](https://github.com/zeromicro/go-zero/pull/4686))

## 📦 Dependency Updates

- Bumped github.com/prometheus/client_golang from 1.21.0 to 1.21.1 ([#4683](https://github.com/zeromicro/go-zero/pull/4683))
- Bumped github.com/fullstorydev/grpcurl from 1.9.2 to 1.9.3 ([#4701](https://github.com/zeromicro/go-zero/pull/4701))
- Bumped github.com/redis/go-redis/v9 from 9.7.1 to 9.7.3 ([#4722](https://github.com/zeromicro/go-zero/pull/4722))
- Bumped github.com/golang-jwt/jwt/v4 from 4.5.1 to 4.5.2 ([#4734](https://github.com/zeromicro/go-zero/pull/4734))
- Bumped github.com/jackc/pgx/v5 from 5.7.2 to 5.7.4 ([#4737](https://github.com/zeromicro/go-zero/pull/4737))

## 🧹 Code Maintenance

- Added more tests and refactoring improvements ([#4677](https://github.com/zeromicro/go-zero/pull/4677), [#4718](https://github.com/zeromicro/go-zero/pull/4718))
