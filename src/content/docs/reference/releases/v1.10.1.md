Released on **2026-03-28** тАФ [GitHub Release](https://github.com/zeromicro/go-zero/releases/tag/v1.10.1)

## ЁЯОЙ Highlights

This patch release adds JSON5 configuration support, generic Redis command execution via `Do`/`DoCtx`, upgrades Go to 1.24, and includes critical security fixes in `core/codec`.

## тЬи New Features

- **core/conf**: Add JSON5 configuration support ([#5433](https://github.com/zeromicro/go-zero/pull/5433))
- **core/stores/redis**: Add `Do`/`DoCtx` for generic command execution ([#5442](https://github.com/zeromicro/go-zero/pull/5442))

## ЁЯРЫ Bug Fixes

- **rest/httpc**: Reject request body for HEAD method in `buildRequest` ([#5457](https://github.com/zeromicro/go-zero/pull/5457))
- **core/codec**: Critical security fixes ([#5479](https://github.com/zeromicro/go-zero/pull/5479))

## ЁЯФз Improvements

- **core**: Replace `TakeOne` usage with `cmp.Or` ([#5461](https://github.com/zeromicro/go-zero/pull/5461))
- **core/stringx**: Replace manual char filter with `strings.Map` ([#5453](https://github.com/zeromicro/go-zero/pull/5453))
- **core/stores/redis**: Reorder `Eval`/`EvalCtx` after `Do`/`DoCtx` for consistent method ordering ([#5502](https://github.com/zeromicro/go-zero/pull/5502))
- **core/mathx**: Add godoc comment to `Numerical` type constraint ([#5470](https://github.com/zeromicro/go-zero/pull/5470))
- Upgrade Go version to 1.24 and update dependencies ([#5499](https://github.com/zeromicro/go-zero/pull/5499))

## ЁЯзк Testing
