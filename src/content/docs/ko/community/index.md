go-zero 커뮤니티에 오신 것을 환영합니다. 이 섹션에서는 기여 방법, 자주 묻는 질문, 커뮤니티 리소스를 안내합니다.

## 기여하기

go-zero는 오픈소스 프로젝트이며 모든 형태의 기여를 환영합니다.

- **버그 보고** — [GitHub issue](https://github.com/zeromicro/go-zero/issues)를 엽니다.
- **버그 수정** — `good first issue` 라벨을 찾아 PR을 보냅니다.
- **기능 추가** — 먼저 issue에서 아이디어를 논의한 뒤 구현합니다.
- **문서 개선** — 빠진 내용이나 오래된 내용을 수정합니다.
- **예제 공유** — [예제](../examples/) 챕터에 추가합니다.

### 시작하기

1. 저장소를 [Fork](https://github.com/zeromicro/go-zero/fork)합니다.
2. fork를 복제합니다: `git clone https://github.com/<you>/go-zero.git`
3. 브랜치를 만듭니다: `git checkout -b fix/typo-in-readme`
4. 변경하고 테스트를 실행합니다: `go test ./...`
5. push한 뒤 Pull Request를 엽니다.
