## 개요

goctl 업그레이드 위한 upgrading goctl tools.

## goctl 업그레이드 directive


```bash
$ goctl upgrade --help
Upgrade goctl to latest version

Usage:
  goctl upgrade [flags]

Flags:
  -h, --help   help for upgrade
```
