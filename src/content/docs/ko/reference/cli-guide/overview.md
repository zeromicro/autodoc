## 개요

goctl은 go-zero와 함께 사용하는 코드 생성 도구입니다. Go HTTP 서비스, Go gRPC 서비스, 데이터베이스 모델, Kubernetes 매니페스트, Dockerfile 등 다양한 산출물을 생성해 반복 작업을 줄여 줍니다.

goctl은 “go control”로 읽습니다. 주요 목적은 개발자가 서비스 골격과 보일러플레이트 코드를 빠르게 만들고 개발 효율을 높이도록 돕는 것입니다.
