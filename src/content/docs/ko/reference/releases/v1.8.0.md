릴리스 날짜: **2025-01-28** — [GitHub 릴리스](https://github.com/zeromicro/go-zero/releases/tag/v1.8.0)

## ✨ Highlights
- Introduced 자동 설정 검증
- Added FreeBSD 지원
- Enhanced 로깅 capabilities 사용하여 새로운 Debugfn과 Infofn 함수
- Added SQL 메트릭 functionality
- Added Redis user 인증 지원

## 🚀 기능
- 지원 위한 HTTP 로 HTTP 에서 gateway
- 자동 설정 검증
- FreeBSD platform 지원
- 새로운 SQL 메트릭 functionality
- Added Debugfn과 Infofn 로 logx/logc
- Redis 인증 사용하여 새로운 User property 에서 RedisConf
- Improved 메트릭 collection 위한 MySQL

## 🐛 버그 수정
- Fixed 헬스 체크 issues, 반환 OK 전에 서버 finishing 시작
- Fixed httpx.ParseJsonBody 오류 사용하여 []byte 필드
- Fixed empty form values handling 에서 HTTP 요청

## 🧹 Chores & Optimizations
- Updated Go 버전
- Multiple 의존성 업데이트합니다:
  - golang.org/x/time 로 v0.9.0
  - golang.org/x/sys 로 v0.29.0
  - google.golang.org/protobuf 로 v1.36.4
  - golang.org/x/net 로 v0.34.0
  - go.mongodb.org/mongo-driver 로 v1.17.2
- Optimized 오류 메시지 위한 mapping 데이터 메서드
- Enhanced 로깅 시스템 사용하여 more 테스트
