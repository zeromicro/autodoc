릴리스 날짜: **2022-04-03** — [GitHub 릴리스](https://github.com/zeromicro/go-zero/releases/tag/v1.3.2)

* CPU 통계에서 `cgroups v2`를 지원합니다.
* `httpc` 패키지를 추가했습니다.
  * HTTP client 호출에 `OpenTelemetry`와 circuit breaker를 활성화했습니다.
  * `httpc.Parse(...)`로 HTTP 응답을 쉽게 parsing할 수 있습니다.
* model tag에서 `-`를 지원해 SQL query에서 해당 field를 무시할 수 있습니다.
* Oracle database에서 `:N` dynamic parameter를 지원합니다.
* Redis `getset` 명령을 지원합니다.
* Redis `SetBit`이 이제 원래 값을 반환합니다.
* `RedisLock`의 re-entrance를 비활성화했습니다.
* 기타 소규모 개선과 버그 수정이 포함되었습니다.
