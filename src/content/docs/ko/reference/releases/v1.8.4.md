릴리스 날짜: **2025-06-08** — [GitHub 릴리스](https://github.com/zeromicro/go-zero/releases/tag/v1.8.4)

## Highlights

- **Continuous profiling** 지원 ([#4867](https://github.com/zeromicro/go-zero/pull/4867)).
   ```yaml
   Profiling:
      ServerAddr: http://<pyroscope-server>:<port>
  ```
- **Embedded 파일 서버 개선 사항**: Added 지원 로 serve 파일 사용하여 `embed.FS`, simplifying static 파일 serving ([#4847](https://github.com/zeromicro/go-zero/pull/4847), [#4851](https://github.com/zeromicro/go-zero/pull/4851)).
- **성능 Optimizations**: Improved 성능 위한 hashing과 slicing operations ([#4891](https://github.com/zeromicro/go-zero/pull/4891), [#4877](https://github.com/zeromicro/go-zero/pull/4877)).

## 변경 사항

- **새 기능**:
    - 파일 서버 now 지원합니다 `embed.FS` ([#4847](https://github.com/zeromicro/go-zero/pull/4847)).
    - Continuous profiling 지원 ([#4867](https://github.com/zeromicro/go-zero/pull/4867)).
- **Optimizations**:
    - Improved slicing과 comparison 사용하여 Go 표준 library 함수 ([#4841](https://github.com/zeromicro/go-zero/pull/4841), [#4840](https://github.com/zeromicro/go-zero/pull/4840), [#4877](https://github.com/zeromicro/go-zero/pull/4877), [#4879](https://github.com/zeromicro/go-zero/pull/4879)).
    - Time formatting standardized 사용하여 Go 표준 library ([#4875](https://github.com/zeromicro/go-zero/pull/4875)).
    - Hashing 성능 개선 ([#4891](https://github.com/zeromicro/go-zero/pull/4891)).
