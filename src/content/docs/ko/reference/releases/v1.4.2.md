릴리스 날짜: **2022-10-22** — [GitHub 릴리스](https://github.com/zeromicro/go-zero/releases/tag/v1.4.2)

## 새로운 기능

- `logc` 패키지를 추가했습니다. `logc.Info(ctx, msg)`처럼 사용할 수 있습니다.
- `logx`에서 debug level을 지원합니다.
- `logx`에서 caller skip을 지원합니다.
- debug log에 color를 추가했습니다.
- `logc`와 `logx`에 `AddGlobalFields`를 추가했습니다.
- `logx`에 `logger.WithFields` 메서드를 추가했습니다.
- `httpx.Parse`에서 string을 map으로 unmarshal할 수 있습니다.
- Redis `blpop`의 timeout을 커스터마이징할 수 있습니다.
- `uuid.UUID`처럼 `encoding.TextUnmarshaler`를 구현한 type의 unmarshal을 지원합니다.
- 지정한 span name에 대해 trace를 비활성화할 수 있습니다.
