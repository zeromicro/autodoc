릴리스 날짜: **2022-03-01** — [GitHub 릴리스](https://github.com/zeromicro/go-zero/releases/tag/v1.3.1)

## 변경 사항

* `go-redis`를 v8로 업데이트하고, Redis 메서드에서 `ctx`를 지원하며, OpenTelemetry를 활성화했습니다. — @kevwan, https://github.com/zeromicro/go-zero/pull/1507
* `sqlx`/`sqlc`에서 `ctx`를 지원하고 OpenTelemetry를 활성화했습니다. — @kevwan, https://github.com/zeromicro/go-zero/pull/1535
* `Cache`에서 `ctx`를 지원하고 OpenTelemetry를 활성화했습니다. — @chenquan, https://github.com/zeromicro/go-zero/pull/1518
* 404 요청을 `traceid`와 함께 기록합니다. — @kevwan, https://github.com/zeromicro/go-zero/pull/1554
* 기타 소규모 버그 수정과 개선이 포함되었습니다.
