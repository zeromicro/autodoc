릴리스 날짜: **2022-07-09** — [GitHub 릴리스](https://github.com/zeromicro/go-zero/releases/tag/v1.3.5)

기능과 업데이트:

1. REST server에 등록된 route를 출력하는 `svr.PrintRoutes()`를 추가했습니다.
2. gRPC error를 HTTP status code로 자동 변환합니다.
3. REST에서 built-in middleware를 비활성화하고 사용자 정의 middleware를 사용할 수 있습니다.
4. `httpc`에 trace-id와 span-id를 추가했습니다.
5. `PeriodLimit`에 `TakeCtx`를 추가했습니다.
6. 기타 소규모 개선과 버그 수정이 포함되었습니다.
