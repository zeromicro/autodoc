릴리스 날짜: **2023-10-28** — [GitHub 릴리스](https://github.com/zeromicro/go-zero/releases/tag/v1.6.0)

## 변경 사항
* upgraded 로 Go 1.19, 때문에 gRPC fixed 보안 issue,과 supported Go 버전 is 1.19+
* feat: 지원 signal SIGINT 위한 graceful shutdown 통해 @Suyghur 에서 https://github.com/zeromicro/go-zero/pull/3611
* feat: 지원 타임아웃 위한 individual rpc 메서드 통해 @vankillua 에서 https://github.com/zeromicro/go-zero/pull/2742
* feat: 모델 추가 withSession 통해 @MarkJoyMa 에서 https://github.com/zeromicro/go-zero/pull/3658
* feat: 지원 위한 비활성화 mon 로그 like sqlx 통해 @shenbaise9527 에서 https://github.com/zeromicro/go-zero/pull/3606
* feat: added `Dec()`과 `Sub()` 에서 `GaugeVec` 에서 메트릭 통해 @Suyghur 에서 https://github.com/zeromicro/go-zero/pull/3666
* minor 버그 수정과 improvements
