## 개요

go-zero의 HTTP 구성 요소인 `rest`는 미들웨어, 요청 검증, JWT 인증 등을 지원하는 고성능 HTTP 서버 기능을 제공합니다.

## 섹션

- [**서버**](server/) — HTTP 서버 설정과 확장
- [**클라이언트**](client/) — 서비스에서 HTTP 요청 보내기
- [기본 서비스](basic/) — 최소 구성의 HTTP 서버 예제
- [JWT 인증](jwt-auth/) — JWT로 경로 보호하기
- [파일 업로드](file-upload/) — 멀티파트 파일 업로드 처리하기
