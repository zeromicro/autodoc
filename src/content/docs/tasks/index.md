Practical, task-oriented guides for working with go-zero. Each guide walks you through a specific task from start to finish.

## Installation

Set up your development environment: install Go, goctl, protoc, and IDE plugins.

## Project Creation

Create a new go-zero project from the command line, GoLand, or VS Code.

## HTTP Service

Build and run a basic HTTP service using goctl and go-zero.

## gRPC Service

Define protobuf schemas, generate code, and run a gRPC server and client.

## CLI Tasks

Use the `goctl` tool: generate API/RPC code, format `.api` files, scaffold MongoDB and MySQL models.

## DSL

Work with the go-zero `.api` and `.proto` DSL syntax.

## Data Storage

Connect to and query MySQL, MongoDB, and Redis.

## Queues

Work with message queues and delay queues.

## Logging

Configure and use go-zero's structured logging.

## Static Configuration

Load and manage static configuration files.

## Memory Cache

Use in-process memory caching.

## Config Center

Integrate with a remote configuration center.
